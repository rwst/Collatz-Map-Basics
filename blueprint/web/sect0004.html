Lemma 21
✓
#
Lean declarations:
  • cycle_implies_not_collatz

If there exists a natural number \(n {\gt} 4\) and \(k \ge 1\) such that \(C^k(n) = n\), then the Collatz conjecture is false (No-cycle condition).

Proof ▶

Suppose \(C^k(n) = n\) for \(n {\gt} 4\). If the conjecture were true, \(n\) would eventually reach \(1\). However, the cycle containing \(n\) consists of values all greater than \(4\) (since \(n \in \{ 1, 2, 4\} \) are the only known cycle values \(\le 4\)), which contradicts reaching \(1\).

Lemma 22
✓
#
Lean declarations:
  • unbounded_orbit_implies_not_collatz

If there exists a natural number \(n\) such that its orbit under \(C\) is unbounded, then the Collatz conjecture is false (No-unbounded-orbit condition).

Proof ▶

If an orbit is unbounded, it cannot reach \(1\), as reaching \(1\) would imply the orbit eventually enters the finite cycle \(\{ 1, 2, 4\} \), making it bounded.

Lemma 23
✓
#
Lean declarations:
  • bounded_no_cycle_implies_collatz

If no natural number \(n {\gt} 4\) lies on a nontrivial cycle and every orbit under \(C\) is bounded, then the Collatz conjecture is true.

Proof ▶

For any \(m \ge 1\), the orbit is bounded, so by the pigeonhole principle it must eventually repeat. Since there are no nontrivial cycles for \(n {\gt} 4\), the orbit must eventually enter the cycle \(\{ 1, 2, 4\} \), thus reaching \(1\).
